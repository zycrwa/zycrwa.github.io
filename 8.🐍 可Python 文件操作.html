🐍 可Python 文件操作

Python 提供了一些内置函数用于文件操作，比如读取和写入文件。让我们一起来看看吧！

1. 打开文件

使用 open() 函数来打开文件。可以选择不同的模式：

  • 'r'：只读（默认）
  • 'w'：写入（会覆盖文件）
  • 'a'：追加（在文件末尾写入）
  • 'b'：二进制模式

2. 读取文件

有几种方法可以读取文件内容：

  • read()：读取整个文件
  • readline()：读取一行
  • readlines()：读取所有行并返回一个列表

总结

  • 打开文件 使用 open() 函数。
  • 读取文件 使用 read()、readline() 或 readlines()。
  • 写入文件 使用 write() 或 writelines()。
  • 追加内容 使用 'a' 模式。
  • 处理二进制文件 使用 'rb' 和 'wb' 模式。
  • 异常处理 确保文件正确关闭。
