🐍 Python 条件语句

条件语句在编程中非常重要，它们允许程序根据不同的条件执行不同的代码块。让我们深入了解 Python 中的条件语句吧！🎉

if 语句 🌟

基本用法

if 语句用来判断一个条件是否为真。如果条件为真，执行相应的代码块。

应用实例：判断成绩是否合格

if-else 语句 🍎

基本用法

if-else 语句用于在条件为假时执行另一段代码。

应用实例：根据温度穿衣建议

if-elif-else 语句 🍉

基本用法

if-elif-else 语句用于处理多个条件，每个条件用 elif 表示，最后用 else 处理所有其他情况。

应用实例：根据时间段问候

嵌套条件语句 🍇

基本用法

条件语句可以嵌套使用，即在一个条件语句中包含另一个条件语句。

应用实例：复杂条件判断

逻辑运算符 🧠

and 逻辑运算符

and 用于同时满足多个条件。

or 逻辑运算符

or 用于满足任意一个条件。

not 逻辑运算符

not 用于取反，将条件的结果反转。

条件表达式（三元运算符） 🍬

条件表达式可以在一行中写出 if-else 语句。

应用实例：根据分数给予评价

综合实例 🌟

实例：根据用户输入判断天气建议

实例：根据用户年龄和是否学生给予优惠

希望这些详细的例子能帮助你更好地理解和使用 Python 中的条件语句！Happy Coding! 💻🎉
