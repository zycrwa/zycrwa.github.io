🐍 Python 函数讲解

函数是一组可重复使用的代码块，它接收输入参数、执行特定任务并返回结果。在 Python 中，函数通过 def 关键字定义。让我们来深入了解 Python 函数吧！🎉

Lambda 函数 🍇

Lambda 函数是一种匿名函数，它可以接收任意数量的参数，但只能有一个表达式，并返回该表达式的结果。

递归函数 🍒

递归函数是指在函数内部调用自身的函数。

作用域 🍑

Python 中有两种作用域：全局作用域和局部作用域。

全局作用域

全局作用域指的是在模块顶层定义的变量和函数。

局部作用域

局部作用域指的是在函数内部定义的变量。

函数文档字符串 🍬

函数文档字符串是对函数进行说明的字符串，位于函数定义的第一行。
