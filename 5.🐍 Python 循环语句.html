🐍 Python 循环语句讲解

循环语句是编程中的重要结构，它允许我们重复执行某段代码，直到满足特定条件。Python 中主要有两种循环：for 循环和 while 循环。让我们详细了解它们的用法吧！🎉

for 循环 🍎

基本用法

for 循环用于遍历序列（如列表、元组、字符串等）中的每个元素。

应用实例：计算列表中所有数字的平方

range 函数 🍊

range 函数生成一个整数序列，常与 for 循环一起使用。

应用实例：计算前 10 个自然数的和

while 循环 🍉

基本用法

while 循环在条件为真时重复执行代码块。

应用实例：计算 1 到 100 的和

break 和 continue 语句 🍇

break 语句

break 语句用于提前退出循环。

应用实例：查找第一个大于 50 的偶数

continue 语句

continue 语句用于跳过当前循环的剩余语句，直接开始下一次循环。

应用实例：计算 1 到 10 中所有奇数的平方

嵌套循环 🍒

基本用法

嵌套循环是指在一个循环体内再包含一个循环。

应用实例：输出一个 3x3 的乘法表

循环与 else 语句 🍍

基本用法

在循环正常结束后，执行 else 语句块。如果循环被 break 语句打断，则不执行 else 语句块。

应用实例：查找元素并反馈

综合实例 🌟

实例：输出一个乘法表

实例：猜数字游戏

希望这些详细的例子能帮助你更好地理解和使用 Python 中的循环语句！Happy Coding! 💻🎉
